## Dense Layer (Layer_Dense) ##

A dense layer (also called fully connected layer) where every input connects to every neuron.
A dense layer performs: Z=X.W+B
This is the core operation of neural networks.

# Why Matrix Multiplication?

Neural networks process many samples efficiently using matrices. Instead of computing neuron outputs one-by-one.
All computations happen simultaneously.
This is vectorization very important for speed.

##  ACTIVATION FUNCITION  ##

# Activation Functions in Neural Networks:

An activation function decides whether a neuron should activate or not.
It transforms the output of a neuron into another value that can be passed to the next layer.
Without activation functions, a neural network becomes only a sequence of linear operations, and it cannot learn complex patterns.

Activation functions introduce:
-> non-linearity--This allows neural networks to learn complex data patterns.

# Types of Activation Functions:

Activation functions are mainly divided into:-
-> Linear Activation Functions
-> Non-Linear Activation Functions

Neural networks mostly use Non-linear activation functions because they help learn complex patterns.

# Common Activation Functions:

Function                        Mostly Used In
--------                        --------------
Binary Step                     Early perceptrons
Linear                          Regression output layers
Sigmoid                         Binary classification
Tanh                            Hidden layers (older networks)
ReLU                            Modern hidden layers
Leaky ReLU                      Improved ReLU
ELU                             Deep networks
Softmax                         Multi-class classification

# ReLU (Rectified Linear Unit) Activation Function: 

This is the most important modern activation function.
-> Formula: f(x)=max(0,x)
-> Negative values become zero
-> Positive values remain unchanged

# Dying ReLU Problem
If a neuron always receives negative values:

Output becomes: 0
Gradient also becomes: zero
Neuron stops learning permanently. This is called Dying ReLU.

# Leaky ReLU
Fixes dying ReLU problem. Instead of zero small negative slope is used.

-> Formula: f(x)=max(0.01x,x)

# Softmax Activation Function

Softmax is mainly used in multi-class classification.
It converts raw outputs into probability distribution.

Suppose final layer outputs: [2.1, 1.3, 0.2]

These are called logits or raw scores. They are not probabilities.

Softmax converts them into: [0.62, 0.28, 0.10]
Now values are between 0 and 1
total sum = 1

Perfect for classification.

## Categorical Cross Entropy (CCE) ##
“Categorical” means the output belongs to categories/classes.
“Entropy” in information theory means uncertainty or randomness
Cross Entropy measures how different the predicted probability distribution is from the actual distribution

**Categorical Cross Entropy (CCE) is a loss function** used in Machine Learning and Deep Learning for multi-class classification problems.

The model predicts probabilities for each class, and Categorical Cross Entropy measures **how wrong the predictions are**.

If the prediction is close to the correct answer → loss is small.
If the prediction is far from the correct answer → loss is large. 

Suppose we have 3 classes:

Class | Meaning
---------------
0       Cat
1	    Dog
2	    Bird

The actual image is:
→ Dog

So the correct output becomes:

Cat     Dog     Bird
0       1       0
This is called One-Hot Encoding.

## Vanishing Gradients ##
Vanishing gradients is a problem that can occur when training deep neural networks.

During training, the network updates its weights using backpropagation. The error from the output layer is propagated backward through all the layers using gradients.

If these gradients become extremely small as they move backward, the earlier layers receive almost no learning signal. As a result, their weights barely change.

## Xavier / Glorot Initialization ##

Developed by Xavier Glorot. The idea is to choose weights so that activations neither grow nor shrink too much as they pass through layers.

For a layer with:
n_inputs = number of incoming neurons
n_outputs = number of outgoing neurons

Weights are initialized roughly around:

Var(W)= 2 / (n_inputs + n_outputs)

Commonly used with:   Sigmoid,  Tanh

---------------------------------------------------------------------------------------------------------------------------------

## He Initialization ##

Developed by Kaiming He. Since ReLU turns negative values into zero, Xavier initialization isn't always ideal.
He initialization compensates for this.

Its variance is:

Var(W)= 2 / n_outputs

Usually best for:   ReLU,   Leaky ReLU

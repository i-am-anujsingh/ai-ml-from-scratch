# AI From Scratch

> A collection of machine learning and large language model implementations built from scratch while learning the fundamentals of Artificial Intelligence.

## About

This repository documents my journey of learning Artificial Intelligence by implementing core algorithms instead of relying on high-level frameworks. My goal is to understand how AI systems work internally by building them step by step using Python, NumPy, and PyTorch.

Rather than only using existing libraries, I focus on implementing the mathematical and engineering concepts behind neural networks and transformer-based language models.

# Large Language Model From Scratch

A GPT-style language model implemented while studying the internal architecture of modern Large Language Models.

This project follows the concepts presented in *LLMs from Scratch* by Sebastian Raschka and is intended for educational purposes.

## Goals

Instead of simply using existing LLM APIs, this project focuses on understanding how transformer-based language models work internally.

## Topics Covered

* Tokenization
* Byte Pair Encoding (BPE)
* Embeddings
* Positional Encoding
* Self-Attention
* Multi-Head Attention
* Transformer Blocks
* Language Modeling
* Text Generation
